PR Description Template
----------------------

Title: Short, imperative (e.g. "Add README suggestions analyzer")

Description:

- **What**: Short summary of the change
- **Why**: Why this is needed
- **How**: Brief notes on implementation

Example:

Add README suggestions analyzer

What: Adds an analyzer that checks README length and presence of a Usage section.
Why: Helps projects present better documentation for users and judges.
How: New `analyzers.js` contains `readmeSuggestions` heuristics; tests added.
